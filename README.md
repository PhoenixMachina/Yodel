A Julia router developed by PhoenixMachina

## Install
To install this package, run this command on your julia console:
```
Pkg.clone("https://github.com/PhoenixMachina/Yodel")
```

## Using Yodel
To start using Yodel, you need to add :
```
using Yodel
```
